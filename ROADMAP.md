# 개발자 웹 이력서 프로젝트 로드맵

## 프로젝트 개요

HTML, CSS, JavaScript, Tailwind CSS를 활용한 반응형 개발자 웹 이력서 제작

---

## 1단계: 프로젝트 초기 설정

### 목표
- 프로젝트 기본 구조 생성
- 개발 환경 구축

### 작업 항목
- [ ] 프로젝트 폴더 구조 생성
  ```
  /
  ├── index.html
  ├── css/
  │   └── style.css
  ├── js/
  │   └── main.js
  ├── assets/
  │   ├── images/
  │   └── icons/
  └── README.md
  ```
- [ ] Tailwind CSS 설정 (CDN 또는 npm 방식)
- [ ] 기본 HTML 템플릿 생성

---

## 2단계: 이력서 컨텐츠 구조 설계

### 목표
- 이력서에 포함될 섹션 정의
- 시맨틱 HTML 구조 작성

### 작업 항목
- [ ] **헤더 섹션**
  - 이름 및 직무
  - 연락처 정보 (이메일, 전화번호, GitHub, LinkedIn)
  - 프로필 사진 (선택 사항)

- [ ] **소개(About Me) 섹션**
  - 간단한 자기소개
  - 주요 관심 분야

- [ ] **기술 스택(Skills) 섹션**
  - 프로그래밍 언어
  - 프레임워크 및 라이브러리
  - 도구 및 플랫폼
  - 시각적 표현 (프로그레스 바 또는 뱃지)

- [ ] **경력(Experience) 섹션**
  - 회사명, 직책, 근무 기간
  - 주요 업무 및 성과
  - 타임라인 형식으로 표현

- [ ] **프로젝트(Projects) 섹션**
  - 프로젝트명 및 설명
  - 사용 기술
  - GitHub 링크 또는 데모 링크
  - 썸네일 이미지

- [ ] **학력(Education) 섹션**
  - 학교명, 전공, 졸업 연도

- [ ] **자격증 및 수상(Certifications & Awards) 섹션**

- [ ] **푸터(Footer) 섹션**
  - 저작권 정보
  - SNS 링크

## 기술 스택 상세

### HTML
- 시맨틱 HTML5 태그 활용
- SEO 친화적인 구조

### CSS
- Tailwind CSS 유틸리티 클래스 활용
- 커스텀 CSS (필요 시)
- CSS 애니메이션

### JavaScript
- Vanilla JS (프레임워크 없이)
- 모던 ES6+ 문법
- DOM 조작 및 이벤트 핸들링

### Tailwind CSS
- 유틸리티 우선 접근
- 반응형 디자인 유틸리티
- 커스텀 테마 설정
